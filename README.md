## 🔥 Getting Started

```sh
# clone repo
$ git clone https://github.com/lucaseverest/email-sender.git

# access folder
$ cd email-sender

# Install the dependencies
$ yarn

# database schema migration
$ yarn prisma migrate dev

# Start the project
$ yarn dev
```

The application can be accessed at [`localhost:3000`](http://localhost:3000).
